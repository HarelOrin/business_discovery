# Business Discovery Engine Plan Run

This folder is the project-specific implementation of the reusable planning system template.

## Why This Exists

- template assets stay reusable in `planning-system/guides`, `planning-system/prompts`, and `planning-system/templates`
- this project keeps its own canonical plan state in this run folder

## Canonical Run Files

- `active-contexts/master-memory.md`
- `active-contexts/origin-chat-context.md`
- `plans/product-strategy-plan.md`
- `plans/technical-implementation-plan.md`
- system router (template guide): `../../guides/file-routing-reference.md`

## Working Rule

When planning this project, prefer attaching the files in this folder rather than template examples.
